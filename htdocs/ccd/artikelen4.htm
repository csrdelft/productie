ORGA / RP nummer
ORGA is een afkorting van organisatie. Het ORGA- of RP (Replacement Parts)-nummer is de code waarmee de productiedag berekend kan worden. De ORGA code begon met nummer 0001 op 9 november 1976. Vanaf die dag is het ORGA nummer elke dag met 1 verhoogd. Vanaf 1980 heeft Citroën de viercijferige ORGA code aangebracht in al haar automobielen. Hieronder een overzicht waar de code per type gevonden kan worden.

A-typen (2cv, Dyane, Ami, etc.)
Op het schutbord onder de motorkap vind je aan de passagierskant tal van codes, waaronder het aluminium typeplaatje en de verfcode (ovalen stempel met iets van AC123 of stempel met iets van EWT vanaf 1985). Er is ook een ronde stempel met AM12. De twee cijfers geven het bouwjaar (Année Modèle) aan en moet overeenkomen met de ORGA code die als viercijferig nummer daar ook gevonden kan worden.

AX
Onder de motorkap, binnenkant wielscherm, vlak voor de linker* veerpoot.

BX
Serie 1: onder de motorkap, binnenkant wielscherm, vlak voor de linker* veerpoot. Serie 2 en 3: op de A-stijl, in de deuropening bestuurderskant, vlak onder het scharnier, naast de sticker voor de bandenspanning.

CX
Onder de motorkap, binnenkant wielscherm, vlak voor de linker* veerpot. Kan onder bedrading zitten.

GSA
Onder de motorkap, binnenkant rechter* wielscherm.

Xantia
Op het schutbord onder de motorkap, vlak voor de rechter* veerpoot, vlak onder de voorruit. Daar zie je de kleurcode (letters) en kleiner het ORGA nummer.

XM
In witte inkt geprint op de A-stijl, in de deuropening bestuurderskant.

Visa/C15
Onder de motorkap, binnenkant wielscherm, vlak voor de rechter* veerpot.

* Met links wordt altijd bestuurderskant bedoeld.
